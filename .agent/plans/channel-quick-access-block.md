# План реализации: Блок быстрого доступа к каналам на странице публикации

## Цель
Создать блок каналов на странице редактирования публикации, который позволит:
1. Быстро переходить к настройкам каналов с существующими постами
2. Быстро создавать посты для каналов без постов
3. Убрать старый способ создания постов (кнопка "Add Channel")

## Структура блока

### Расположение
- Под основным блоком публикации (после `PublicationForm`)
- Перед секцией постов

### Визуальная структура
```
┌─────────────────────────────────────────┐
│ Channels                                │
├─────────────────────────────────────────┤
│ [Icon] [Icon] [Icon]  ← Каналы с постами│
│ [Icon] [Icon]         ← Каналы без постов│
└─────────────────────────────────────────┘
```

### Первая строка - Каналы с постами
- Отображаются иконки соц сетей у которых есть посты
- При клике на иконку - скролл к якорю настроек этого канала (пост блок)
- Группировка по socialMedia (если несколько каналов одной соц сети - показываем один раз)

### Вторая строка - Каналы без постов  
- Отображаются иконки соц сетей на которых постов нет
- При клике на иконку - создается пост сразу (не виртуально)
- Группировка по socialMedia

## Технические детали

### Компонент
Создать новый компонент: `ui/app/components/publications/ChannelQuickAccessBlock.vue`

#### Props
```typescript
interface Props {
  publication: PublicationWithRelations
  channels: ChannelWithProject[]
}
```

#### Computed properties
1. `channelsWithPosts` - каналы у которых есть посты в публикации
2. `channelsWithoutPosts` - доступные каналы без постов
3. `socialMediaWithPosts` - уникальные соц сети с постами
4. `socialMediaWithoutPosts` - уникальные соц сети без постов

#### Методы
1. `scrollToPost(channelId)` - скролл к посту канала
2. `createPostForChannel(channelId)` - создание поста для канала

### Изменения в странице edit.vue
1. Добавить новый компонент после блока формы публикации
2. Убрать кнопку "Add Channel" и виртуальное создание постов
3. Убрать `isCreatingPost` ref и связанную логику

### API
Использовать существующий `createPost` из `usePosts` composable

### Стилизация
- Использовать существующие утилиты для иконок соц сетей
- Hover эффекты
- Tooltip с названием канала
- Disabled состояние если нельзя создать пост
