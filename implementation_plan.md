# План реализации рефакторинга страницы редактирования публикации

Этот план описывает изменения в макете страницы редактирования публикации для улучшения UX, согласно запросу пользователя.

## Цели
1. Поменять местами блоки "Статус" и "Проект" в блоке метаданных публикации.
2. Поменять местами левую и правую колонки в блоке метаданных.

## Изменения
### 1. `ui/app/pages/publications/[id]/edit.vue`
- Изменен порядок колонок внутри `div.grid`.
- Старая правая колонка (Проект, Статус, Язык, Тип) перемещена влево.
- Старая левая колонка (Создано, Планирование) перемещена вправо.
- Внутри новой левой колонки изменен порядок блоков:
    - Блок "Статус" теперь идет первым.
    - Блок "Проект" идет вторым.

## Результат
Пользователь увидит обновленный интерфейс, где инструменты управления статусом и проектом находятся слева и в более логичном порядке (Статус -> Проект), а информационные поля (дата создания, планирование) смещены вправо.
